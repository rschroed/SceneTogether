# Design principles reference

SceneTogether should feel:

- cinematic
- decisive
- electric
- intentional

Reference influences:

- dark mobile UI with neon signal accents
- poster-style typography
- high-contrast graphic composition
- controlled punk-flyer energy

Key rules:

- one dominant focal point
- design for convergence
- use bold type and limited color
- keep mobile intentional
- prefer fewer, stronger patterns
