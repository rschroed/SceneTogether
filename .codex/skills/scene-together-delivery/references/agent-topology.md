# Agent topology reference

Default roles:

- Product Planner: frame issues, acceptance criteria, implementation plan
- Delivery Engineer: implement the scoped issue
- Reviewer: independent pass for correctness and regressions
- Issue Steward: maintain issue, branch, PR, and follow-up hygiene

Default sequence:

1. plan
2. implement
3. review
4. update GitHub state

Only introduce more specialist roles when the repo complexity justifies them.
